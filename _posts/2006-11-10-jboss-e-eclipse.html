Lavorando spesso su diversi progetti nel medesimo lasso di tempo, quando uso jboss di norma copio il contenuto della directory $JBOSS_HOME/servers/default in una directory "di progetto"; in questo modo la configurazione per uno specifoco progetto non inquina gli altri.

Purtroppo quando configuri Eclipse per eseguire il server jboss l'elenco prevede i soli server "standard": all, default e minimal.

Per aggiungere un server extra senza andare ad agire sul parametro di esecuzione puoi modificare il file

$ECLIPSE_HOME/plugins/org.eclipse.jst.server.generic.jboss_1.5.1.v200609140551/servers/jboss.serverdef

(ovviamente il numero di revision del plugin pu√≤ variare).
